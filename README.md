# OursPaay
Plataforma de cobranças moderna, segura e escalável. Multilíngue (pt-BR, en, es, ru), acessível (WCAG 2.1 AA), estilo Dashboard Admin. Chat com IA, notificações em tempo real, pagamentos via PIX, cartão, boleto e cripto, carteira digital, cashback, indicações e proteção avançada contra ataques.
